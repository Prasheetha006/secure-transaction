# 🔐 SecureTx - Secure Transaction System

## 📌 Description
SecureTx is a web-based application designed to demonstrate secure financial transactions. It allows users to create, track, and manage transactions with built-in security features and monitoring.

---

## 🚀 Features
- User login system
- Create new transactions
- View transaction history
- Transaction status (Pending / Approved)
- Dashboard with spending overview
- Chatbot support
- Basic fraud detection

---

## 🖥️ Dashboard
- Total spending summary
- Transaction count
- Pending & approved transactions
- Quick actions

---

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- MongoDB / SQLite

---

## 📂 Files
- `safe_advanced.py` → Main app
- `gen_key.py` → Generates secret key
- `secret.key` → Encryption key
- `chat.html` → Chatbot UI
- `requirements.txt` → Dependencies
